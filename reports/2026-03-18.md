# OpenClaw Security Report — 2026-03-18

## Verdict: FAIL

## Summary
The security scan results indicate multiple high and critical severity vulnerabilities in various packages, including @hono/node-server and fast-xml-parser, which can be exploited to bypass authorization and cause denial of service. There are also several low and medium severity misconfigurations in the Dockerfiles.

## Critical Findings
- [HIGH] @hono/node-server via npm audit
  Issue: Authorization bypass for protected static paths via encoded slashes in Serve Static Middleware
  Fix: npm audit fix --force
- [HIGH] fast-xml-parser via npm audit
  Issue: Numeric entity expansion bypassing all entity expansion limits
  Fix: npm audit fix --force
- [CRITICAL] fast-xml-parser via Trivy
  Issue: Cross-Site Scripting (XSS) due to improper DOCTYPE entity handling
  Fix: Update to version 5.3.5 or 4.5.4

## Warnings
- Multiple high severity vulnerabilities in @angular/compiler and @angular/core
- Multiple medium severity misconfigurations in Dockerfiles

## Recommendation
Due to the presence of high and critical severity vulnerabilities, it is not recommended to deploy this code without first addressing these issues. The available fixes should be applied by running 'npm audit fix --force' to update the vulnerable packages. Additionally, the low and medium severity misconfigurations in the Dockerfiles should be reviewed and addressed to ensure the security and integrity of the application. It is also recommended to regularly run security scans and audits to identify and address any new vulnerabilities that may arise.
