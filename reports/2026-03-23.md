# OpenClaw Security Report — 2026-03-23

## Verdict: FAIL

## Summary
The scan results indicate the presence of HIGH and CRITICAL severity vulnerabilities in the dependencies of the project, with available fixes for some of them. Specifically, the fast-xml-parser library has a CRITICAL vulnerability that can be fixed by updating to version 5.3.5 or 4.5.4. Additionally, there are several HIGH severity vulnerabilities in the @angular/compiler and @angular/core libraries that can be fixed by updating to the specified versions.

## Critical Findings
- [CRITICAL] fast-xml-parser via Trivy
  Issue: Cross-Site Scripting (XSS) due to improper DOCTYPE entity handling
  Fix: Update to version 5.3.5 or 4.5.4

## Warnings
- Medium severity misconfiguration in Dockerfile
- Low severity misconfigurations in Dockerfile.sandbox
- Dockerfile.sandbox-browser
- Dockerfile.sandbox-common

## Recommendation
The project should not be deployed in its current state due to the presence of CRITICAL and HIGH severity vulnerabilities. The dependencies should be updated to the latest versions to fix these vulnerabilities. Specifically, the fast-xml-parser library should be updated to version 5.3.5 or 4.5.4, and the @angular/compiler and @angular/core libraries should be updated to the specified versions. Additionally, the medium and low severity misconfigurations in the Dockerfiles should be addressed by adding a tag in the 'FROM' statement and a HEALTHCHECK instruction. Only after these issues are resolved can the project be considered secure for deployment.
