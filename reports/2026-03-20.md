# OpenClaw Security Report — 2026-03-20

## Verdict: FAIL

## Summary
The scan results reveal multiple critical and high-severity vulnerabilities in various packages, including fast-xml-parser and node-forge, which can lead to Cross-Site Scripting (XSS), Denial of Service, and other security issues. The vulnerabilities are present in both the npm and pnpm packages. Additionally, there are some misconfigurations in the Dockerfiles that can be improved for better security.

## Critical Findings
- [CRITICAL] fast-xml-parser via Trivy
  Issue: Cross-Site Scripting (XSS) due to improper DOCTYPE entity handling
  Fix: Update to version 5.3.5 or 4.5.4

## Warnings
- Misconfigurations in Dockerfiles
- High-severity vulnerabilities in node-forge package

## Recommendation
Given the presence of critical and high-severity vulnerabilities with available fixes, it is recommended to update the affected packages to the latest secure versions. Specifically, update fast-xml-parser to version 5.3.5 or 4.5.4. Additionally, address the misconfigurations in the Dockerfiles by adding HEALTHCHECK instructions. It is not recommended to deploy the code in its current state due to the potential security risks. Instead, prioritize fixing these vulnerabilities and misconfigurations before deployment.
