# OpenClaw Security Report — 2026-04-08

## Verdict: FAIL

## Summary
The security scan results indicate the presence of HIGH and CRITICAL severity vulnerabilities in the project's dependencies, specifically in the `fast-xml-parser` and `handlebars` packages. These vulnerabilities can lead to security issues such as Cross-Site Scripting (XSS) and Denial of Service (DoS) attacks. A total of 5 HIGH and 2 CRITICAL vulnerabilities were found in the `vendor/a2ui/renderers/angular/package-lock.json` and `vendor/a2ui/specification/0.8/eval/pnpm-lock.yaml` files.

## Critical Findings
- [CRITICAL] fast-xml-parser via Trivy
  Issue: CVE-2026-25896: Cross-Site Scripting (XSS) due to improper DOCTYPE entity handling
  Fix: Update to version 5.3.5 or 4.5.4
- [CRITICAL] handlebars via Trivy
  Issue: CVE-2026-33937: Remote Code Execution via crafted Abstract Syntax Tree object
  Fix: Update to version 4.7.9

## Warnings
- DS-0001: Specify a tag in the 'FROM' statement for image 'base'
- DS-0026: Add HEALTHCHECK instruction in your Dockerfile

## Recommendation
Based on the security scan results, it is strongly recommended to update the vulnerable dependencies to the latest versions to prevent potential security issues. Specifically, update `fast-xml-parser` to version 5.3.5 or 4.5.4 and `handlebars` to version 4.7.9. Additionally, address the misconfiguration issues by specifying a tag in the 'FROM' statement for the 'base' image and adding a HEALTHCHECK instruction in the Dockerfile. It is also recommended to regularly review and update dependencies to ensure the project's security posture is maintained.
